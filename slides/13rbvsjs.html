Ruby vs JS

Ruby

sum(“2”, 2) #=>  TypeError: String can't be coerced into Fixnum

JavaScript

sum(“2”, 2) #=> “22”
Ruby isn’t as loosely typed as JavaScript, for contrast. *If we called sum(“2”, 2) #=> TypeError: String can't be coerced into Fixnum we’d get the error above, *but in JS: sum(“2”, 2) #=> “22”
